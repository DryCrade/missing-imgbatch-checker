Missing IMGBatch Checker



Find out what IMG_xxxx is missing from a folder

Step-by-step tutorial

1. Choose the folder you want to scan.

2. Click on 'Upload'. The website should display all the file names that are uplaoded.
NOTE: Do not skip this step or it will not work.

3. Declare the range you want to scan.
EXAMPLE: If you want to start scanning from IMG_1000 to IMG_2000.
Set the value of 'from' to 1000 and 'to' to 2000.

4. Click 'Check' and enjoy the results!
